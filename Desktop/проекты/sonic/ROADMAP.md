## Следующий шаг — подключить Supabase

1. Создать проект на [supabase.com](https://supabase.com)
2. Скопировать `Project URL` и `anon key` → вставить в `.env.local`
3. В `SQL Editor` запустить миграции по порядку: `0001_schema.sql` → `0002_rls.sql` → `0003_seed.sql`
4. В `Authentication → Users` создать 3 юзера, вставить UID в `0003_seed.sql` и запустить INSERT
5. В `Database → Replication` включить таблицы `rooms` и `sessions`
6. `npm run dev` → готово
